🔗 การเชื่อมต่อ Google Sheets

ไม่ได้เชื่อมต่อ
📋 คัดลอก URL จาก Google Sheets ที่ต้องการใช้เป็นฐานข้อมูล
🛡️ สร้าง API Key ได้ที่ Google Cloud Console
📝 ชื่อของ Sheet ที่จะใช้เก็บข้อมูล (ค่าเริ่มต้น: Prospects)
📡 ฐานข้อมูลแบบเรียลไทม์:
  • ข้อมูลทั้งหมดเก็บใน Google Sheets
  • ระบบดึงข้อมูลแบบเรียลไทม์ทุก 30 วินาที
  • ทีมสามารถแก้ไขข้อมูลใน Google Sheets ได้โดยตรง
  • การเปลี่ยนแปลงจะปรากฏในระบบทันที
  • สามารถทำงานร่วมกันได้หลายคนพร้อมกัน
📚 วิธีการตั้งค่า Google Sheets:
  1. เข้าไปที่ Google Cloud Console
  2. สร้างโปรเจ็กต์ใหม่หรือเลือกโปรเจ็กต์ที่มี
  3. เปิดใช้งาน Google Sheets API
  4. สร้าง API Key ในส่วน Credentials
  5. คัดลอก URL ของ Google Sheets ที่ต้องการใช้
  6. ตั้งค่าสิทธิ์ของ Google Sheets เป็น "Anyone with the link can edit"
